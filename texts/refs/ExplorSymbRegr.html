Exploração de “Regressão Polinomial com Algoritmos Genéticos”

É um percurso já percorrido (ver nas referências, abaixo). No entanto, pode ser uma experiência interessante. O “R” tem duas bibliotecas para algoritmos genéticos, rpg e rgenoud. Destas, embora a segunda seja bastante usada, a primeira é orientada para “Symbolic Regression” como se pode ver nos tutoriais do wiki do rgp.

Nos tutoriais encontramos o seguinte exemplo:

library(rgp)

## Loading required package: emoa

## Loading required package: rrules


# Define the observations
x1 <- seq(0, 4 * pi, length.out = 201)
x2 <- seq(0, 4 * pi, length.out = 201)

# a latent variable, with a trick expression and some noise
y <- 2 * sin(x1) + cos(2 * x2) + 0.1 * x1 * x2 + 0.1 * rnorm(201)

# Caution: there are **ONLY** 201 observations (and not 201*201)
observations <- data.frame(y = y, x1 = x1, x2 = x2)

# Define which functions can be used to construct the candidates
validFunctions <- functionSet("+", "sin", "*", "cos")

# Search
result <- symbolicRegression(y ~ x1 + x2, data = observations, functionSet = validFunctions, 
    stopCondition = makeStepsStopCondition(1000), verbose = FALSE)

# Show the real observations (in RED)
plot(observations$y, col = "red", type = "l", xlab = "index", ylab = "y (latent)", 
    main = "Symbolic Regression")

legend("topleft", c("real", "predicted"), col = c("red", "black"), lty = c(1, 
    1))
# And the predictions (in BLACK)
points(predict(result, newdata = observations), col = "black", type = "l")
